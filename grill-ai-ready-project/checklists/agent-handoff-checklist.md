# AI Agent Handoff Checklist

Before switching sessions, agents, or models:

- [ ] Update `docs/PROJECT_STATE.md`
- [ ] Add current objective
- [ ] Add completed work
- [ ] Add in-progress work
- [ ] Add pending tasks
- [ ] Add files to read first
- [ ] Add files likely to change
- [ ] Add files/areas not to touch
- [ ] Add validation commands
- [ ] Add open questions
- [ ] Add known risks
