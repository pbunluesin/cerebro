# Security Readiness Checklist

- [ ] `.env.example` contains placeholders only
- [ ] Real `.env` files are ignored/protected
- [ ] Service account JSON / credentials are ignored/protected
- [ ] Auth flow documented
- [ ] Authorization rules documented
- [ ] Sensitive data identified
- [ ] Logs do not expose secrets or unnecessary PII
- [ ] Input validation documented
- [ ] Dependency vulnerability check exists or is planned
- [ ] Production permissions are least-privilege where possible
