# Prompt: Codex Review

Review the current diff as an independent reviewer.

Do not modify code unless explicitly requested.

Focus on:

- Correctness
- Security
- Data integrity
- API contract compatibility
- Database semantics
- Business rule preservation
- Maintainability
- Test coverage
- Observability impact
- Deployment risk

Output format:

## Critical
- [Finding]

## High
- [Finding]

## Medium
- [Finding]

## Low
- [Finding]

## Suggested Fix Plan
Explain what Claude should fix.
