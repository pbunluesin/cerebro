# Project Bootstrap Skill

## Purpose

Use this skill to initialize a new AI-agent-ready software project.

## Steps

1. Read repository structure.
2. Identify stack, database, integrations, and deployment environment.
3. Generate or update:
   - AGENTS.md
   - CLAUDE.md
   - PROCESS.md
   - PROJECT_STATE.md
   - docs/
4. Keep instruction files concise.
5. Put deep knowledge in docs/.
6. Ask for missing context only when necessary.

## Output

Return:

- Files created/updated
- Assumptions
- Risks
- Next recommended steps
